# Codex Extensions Paperback 0.8

This folder is a combined Paperback 0.8 repository which includes:

- `HiveToons`
- `VortexScans`

## Layout

- `versioning.json`: shared Paperback repo manifest
- `index.html`: shared landing page with one Add to Paperback link
- `HiveToons/source.js`: HiveToons source
- `VortexScans/source.js`: Vortex Scans source

## How to use it

1. Host the contents of this folder on one static URL.
2. Add that one repo URL to Paperback.
3. Paperback will then show both sources from the same repo.

## GitHub Pages

If you use GitHub Pages, upload the contents of this folder to the repo root so the published site looks like this:

- `/versioning.json`
- `/index.html`
- `/HiveToons/source.js`
- `/HiveToons/includes/icon.webp`
- `/VortexScans/source.js`
- `/VortexScans/includes/icon.webp`
